## Getting Started

#Install the the necessary dependencies by doing
```yarn or npm i```

#Add environment variable
Obtain the API_KEY from openweather and create a `.env.local`, and then add `API_KEY=XXXX` 

#Run the development server:
```bash
npm run dev
# or
yarn dev
```
